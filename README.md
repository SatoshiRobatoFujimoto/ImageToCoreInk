# ImageToCoreInk
Image Converter for M5Stack CoreInk

## Download binary
- [win10-x86]()
- [osx-x64]()
- [linux-x64]()

## Usage
1. Convert image file(s) to header file.
```
ImageToCoreInk [image_file|image_folder]
```

2. Include the header file to your Arduino project.
- [example](sample/arudino/sample_image.ino)
